1. Media queries

    Task: Create a 100px by 100px div. Write @media queries for this div such that

    • When screen width is below 500px, the background color for the div is red
    • When screen width is between 500px and 1000px, the background color for the div is green
    • When screen width is above 1000px, the background color for the div is blue

    Hint: If you want to easily test out media queries by changing the width of the page then try figuring out how to open the "Responsive Design Mode" for your browser
