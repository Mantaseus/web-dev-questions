  6. CSS Animations - Jiggle on click

    Task: Make a 50px by 50px circle with background color of #ddd. Make it animate roughly as shown below while the user clicks on the circle (i.e use :active). The animation is 1 second long. Use ease transition pacing. Here are the keyframes

    • 0% - scale X=1, Y=1
    • 30% - scale X=1.25, Y=0.75
    • 40% - scale X=0.75, Y=1.25
    • 50% - scale X=1.15, Y=0.85
    • 65% - scale X=0.95, Y=1.05
    • 75% - scale X=1.05, Y=0.95
    • 100% - scale X=1, Y=1

    Try playing around with the time of the animation and notice how it can make it seem as if the circle is very springy or jelly-like

    Notice the scaling values. The X and Y values always add up to 2. When making animations like this it is important to remember the "Law of conservation of Mass". If the mass is not conserved then the animation can look a bit unnatural (but sometimes you might want that)
