  2. Padding: With a single CSS property

    Task: Define a div withing a div as shown below 

              <div class="container-2" style="background-color: red; width: 100px; height: 100px;">
            <div style="background-color: blue; width: 100%; height: 100%;"></div>
          </div>

    Define CSS for the container-2 class using just the padding CSS property only such that the top space is 5px, right space is 10px, bottom space is 15px, and left space is 20px. The format would look something like this (figure out what to put in the blanks) 

              .container-2 {
            padding: __ __ __ __;
          }

    Note: This is a convenient format that allows us to define the padding for multiple directions in a single line
